# Stuffbucket Homebrew Tap

A [Homebrew tap](https://docs.brew.sh/Taps) providing custom formulas for stuffbucket tools and utilities. This repository makes it easy to install and manage stuffbucket software using Homebrew, the largest macOS package manager in the world.

New to Homebrew? [Get started here](HOMEBREW.md).

## Manual Installation

All formulas can be installed directly:

```bash
brew tap stuffbucket/tap
brew install stuffbucket/tap/<formula-name>
```

Homebrew automatically taps this repository when you install a formula this way.
