Welcome to stuffbucket Lima Installer

This installer will set up Lima from the stuffbucket tap on your system.

Lima is a Linux virtual machine manager with desktop GUI support.

What will be installed:

  • Lima virtual machine manager (v2.0.0-beta.0.1-fork)
  • stuffbucket/tap Homebrew repository

Requirements:

  • Homebrew must be installed (install stuffbucket Homebrew package first)
