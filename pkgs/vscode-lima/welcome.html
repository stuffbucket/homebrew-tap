Welcome to stuffbucket VS Code Lima Extension Installer

This installer will set up the VS Code Lima Manager extension on your system.

The Lima Manager extension provides a graphical interface for managing Lima virtual machines in VS Code.

What will be installed:

  • vscode-lima command-line installer
  • Lima (if not already installed)
  • stuffbucket/tap Homebrew repository

Requirements:

  • Homebrew must be installed
  • Visual Studio Code must be installed to use the extension
